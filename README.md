# Azmi Muwahid — Building the next generation of Agentic AI

I’m a Senior Full Stack Engineer with a 6-year track record of shipping scalable web apps. These days, I’m obsessed with moving beyond simple chat interfaces. I specialize in building Agentic Workflows, RAG architectures, and low-latency AI Audio streaming—essentially, making AI systems that don't just talk, but actually act and reason in real-time.

## 🚀 The AI Edge
- **Agentic Workflows**: Multi-agent orchestration and autonomous tool-calling.
- **RAG Architectures**: High-precision retrieval with vector databases.
- **Real-Time Layer**: Binary audio streaming and live state management.
- **Battle-Tested Infra**: AWS (ECS/Lambda/S3), Google Cloud, Terraform.

## 🛠️ Tech Stack
- **Languages**: TypeScript, Ruby, Go, Python.
- **Frontend**: SvelteKit, React, Next.js, Tailwind CSS.
- **Backend**: Ruby on Rails, tRPC, Node.js.
- **Database**: PostgreSQL, Redis, Pinecone, Elasticsearch.

## 📦 Project Structure
- `src/lib/components/sections`: Production-grade portfolio sections.
- `src/lib/actions`: Svelte actions for animations and interactions (e.g., `inview`).
- `static/`: High-resolution assets including the new Agentic AI logo.

## 🛠️ Development
